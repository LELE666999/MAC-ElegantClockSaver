# ElegantClockSaver (优化版)

## 简介
这是一个为 macOS 设计的优雅时钟屏保。
本次更新优化了代码结构和用户界面，专注于提供极致的文字和粒子时钟体验。

![设置界面预览](Frame%209.png)

### 主要变更
- **精简主题**：移除了史努比和翻页时钟主题，保留了核心的“字体”、“粒子”和“倒计时”模式。
- **UI 优化**：重新设计了设置界面，布局更紧凑，分类更清晰（外观、显示、位置、颜色）。
- **性能提升**：清理了冗余代码，运行更轻量。

## 安装方法
1. 找到文件夹中的 `ElegantClockSaver_Optimized.saver` 文件。
2. **双击打开**：系统会提示安装，选择“为该用户安装”或“为所有用户安装”。
3. 或者 **手动安装**：
   - 打开“访达” (Finder)。
   - 按 `Cmd + Shift + G`，输入 `~/Library/Screen Savers/` 并回车。
   - 将 `.saver` 文件拖入该文件夹。
4. 打开 **系统设置** -> **屏幕保护程序**，在列表中选择 `ElegantClockSaver`。
5. 点击“选项...”可自定义字体、颜色、动画等。

## 支持的主题

### 1. 字体主题
支持多种自定义字体，可调节大小、间距、动画效果。

# ElegantClockSaver (optimized version)

## Brief introduction

This is an elegant clock screen saver designed for macOS.

This update optimizes the code structure and user interface, focusing on providing the ultimate text and particle clock experience.

![Settings Interface Preview](Frame%209.png)

### Major changes

- **Simplified theme**: Snoopy and page-turning clock themes have been removed, and the core "font", "particle" and "countdown" modes have been retained.

- **UI Optimization**: Redesigned the setting interface, with a more compact layout and clearer classification (appearance, display, position, color).

- **Performance improvement**: Redundant code has been cleaned up and the operation is lighter.

## Installation method

1. Find the `ElegantClockSaver_Optimized.saver` file in the folder.

2. **Double-click to open**: The system will prompt you to install, select "Install for this user" or "Install for all users".

3. Or **manual installation**:

- Open Finder.

- Press `Cmd + Shift + G`, enter `~/Library/Screen Savers/` and press Enter.

- Drag the `.saver` file into the folder.

4. Open **System Settings** -> **Screen Saver** and select `ElegantClockSaver` in the list.

5. Click "Options..." Fonts, colors, animations, etc. can be customized.
